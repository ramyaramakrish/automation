How to install Maven?
-----------------------------    

on windows
-----------------------------
1. Download the Maven binary zip archive from the official Apache Maven website: https://maven.apache.org/download.cgi
2. Extract the downloaded zip file to a directory of your choice 
    (e.g., C:\Program Files\Apache\Maven).
3. Set the M2_HOME environment variable to point to the Maven installation directory.
    - Right-click on 'This PC' or 'Computer' on the desktop or in File Explorer.
    - Select 'Properties'.
    - Click on 'Advanced system settings' on the left side.
    - Click on the 'Environment Variables' button.
    - Under 'System variables', click 'New' and enter:
        Variable name: M2_HOME
        Variable value: C:\Program Files\Apache\Maven (or your installation path)
4. Add the Maven 'bin' directory to the PATH environment variable.
    - In the 'Environment Variables' window, find the 'Path' variable under 'System variables' and select it.
    - Click 'Edit', then 'New', and add the path to the Maven 'bin' directory:
        C:\Program Files\Apache\Maven\bin
5. Click 'OK' to close all dialog boxes.
6. Open a new Command Prompt window and type `mvn -version` to verify the installation. You should see the Maven version information displayed.
